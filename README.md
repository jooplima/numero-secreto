# Descrição do projeto
Projeto desenvolvido para o curso de introdução a linguagem de programação do curso da Alura. O Jogo do número secreto é um pequeno jogo de "descubra o número" em que o jogador insere um número entre 1 e 100 e a máquina responde se ele descobriu o número ou não. Tente sua sorte e veja se consegue descobrir o número com apenas uma tentativa!
